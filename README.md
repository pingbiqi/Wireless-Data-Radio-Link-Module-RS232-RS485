为了方便工程师评估数传电台的通讯距离，我们提供了一个基于 Python 的链路预算计算脚本。

### 理论模型 (Theoretical Model)
接收功率计算公式：
$$P_{rx} = P_{tx} + G_{tx} + G_{rx} - L_{path} - L_{misc}$$

* $P_{rx}$: 接收功率 (dBm)
* $P_{tx}$: 发射功率 (dBm)
* $G_{tx}/G_{rx}$: 天线增益 (dBi)
* $L_{path}$: 路径损耗 (dB)
